# ukm_vest ESX Script

A lightweight and configurable ESX script that allows players to equip and unequip clothing items such as safety vests.  
This resource uses ESX usable items, clothing components and proper equip and unequip animations.  
Fully extendable for any item you want to toggle via the inventory.

## Features

- ESX usable item support
- Equip and unequip clothing with real animations
- Configurable clothing components, drawables and textures
- Works with any item (example: safety vest)
- Easy to extend, clean structure
- Notification feedback via ESX

## Requirements

- ESX (es_extended)
- A FiveM Server running the latest recommended artifacts
